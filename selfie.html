Selfie Automation using Open CV

Haar Cascade is an ML object detection algorithm used to identify objects in an image (treated as a matrix i.e. 2D grid here) or video. In this algorithm, a cascade function is trained from a lot of positive and negative images which is then used to detect objects in other images.

It can be trained to identify almost any object.

In this project, we will be using these pre-trained files.
